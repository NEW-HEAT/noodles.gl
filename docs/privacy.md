# Privacy & Telemetry Notice

Noodles.gl uses limited, privacy-preserving analytics through Posthog to understand aggregate usage patterns and improve the open-source project.

## What We Collect

- **Aggregate event data:** Page views, feature usage, and interaction patterns
- **No personally identifiable information (PII):** We never collect names, email addresses, IP addresses, or any personal data
- **No project data:** Your projects, node content, code, SQL queries, and visualizations are never collected or transmitted

## How We Use Data

- **Improve the project:** Understand which features are most useful to prioritize development
- **Not for commercial use:** Data is not used by Joby Aviation for marketing, advertising, or any commercial purposes
- **Open source benefit:** Insights help make Noodles.gl better for the entire community

## Your Control

You have full control over analytics tracking:

1. **Consent prompt:** When you first visit Noodles.gl, you'll see a consent banner asking for permission
2. **Change anytime:** You can update your consent preference in the Settings dialog at any time
3. **No functionality impact:** Declining analytics does not affect any features or capabilities
