# Privacy Policy for Noodles.gl

Noodles.gl is an open-source visualization tool developed under the Apache 2.0 License and sponsored by Joby Aviation.

We use Posthog to collect anonymized, aggregate analytics on site usage to help improve functionality and user experience. We present a privacy-preserving approach to analytics, ensuring that no personally identifiable information (PII) or project-specific data is collected. Users are given the choice to opt-in or opt-out of analytics tracking.

## Data Collection

- **Data collected:** Aggregate event data (e.g., page views, feature usage), no personally identifiable information (PII).
- **Error reports:** Error reports are sent by default to help us identify and fix bugs. Error reports contain only technical information (error messages and stack traces) and do not include any personal data or project content. You can disable error reporting in the Settings dialog.
- **What we don't collect:** We never collect your project data, node content, code, SQL queries, or any personal information.

## Data Usage and Sharing

- **Data sharing:** Analytics are not used by Joby Aviation or any third party for advertising, profiling, or commercial use.
- **Purpose:** Data is used solely to understand which features are most useful and improve the open-source project.

## Hosting and Third-Party Services

- **Hosting:** The app is hosted via GitHub Pages; GitHub may collect standard server logs per its own privacy policy.
- **Analytics provider:** We use Posthog for privacy-preserving analytics. You can opt out at any time in the application settings.

## Your Choices

- You can accept or decline analytics tracking when you first visit the application.
- You can change your consent preference at any time in the Settings dialog.
- Declining analytics does not affect any functionality of Noodles.gl.

## Contact

For questions or concerns about privacy, please open an issue in [our GitHub repository](https://github.com/joby-aviation/noodles.gl/issues).

## Scope

This policy applies only to the hosted version of Noodles.gl at [https://noodles.gl](https://noodles.gl) and the web application at [https://app.noodles.gl](https://app.noodles.gl).
